⟡ Common Issues

Xeno not opening

You do not have the required dependencies downloaded. Download Microsoft Visual C++ Redistributable and .NET Runtime 8.0.0

  • → Microsoft Visual C++ Redistributable
  • → .NET Runtime 8.0.0

Script not executing

There was an interference attaching to Roblox. Restart Roblox and Xeno and try again; reattach to resume execution.
